Global.helma.Html.TableWriter (numberOfColumns, attr)
Creates a new TableWriter instance

  • Parameter {Number}:
    numberOfColumns The number of columns in the table
  • Parameter {Object}:
    attr An object containing attributes to use when rendering the single table elements. For a description see {@link #table}.
  • Returns
    An instance of TableWriter
